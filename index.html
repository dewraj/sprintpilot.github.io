Sprint Pilot

Generate professional sprint reports from Azure DevOps — no terminal knowledge required.

Sprint Pilot helps teams generate polished sprint reports (planning, closure, productivity, and risk) and send them to stakeholders via Outlook on Windows.

Quick Start (No Terminal)

  1. Setup (one time): double-click setup.cmd
  2. Launch: double-click sprintpilot-launcher.cmd (opens http://localhost:3000)
  3. Generate: select your team, choose a report type, preview, and send
Tip: The README includes a full step-by-step guide with screenshots (Download ZIP → Extract to C:\ → Setup → Azure login → Launch).

Configuration

config.settings.json is pre-configured and uses your Azure DevOps SSO access (authenticated during setup). You only need to edit it to customize team settings or area paths.

Advanced: Command-Line Tools

For automation and power users, run reports directly: 

npm run sprint-report          # Sprint summary
npm run sprint-report-full     # Full sprint summary
npm run risk-report            # Risk analysis
npm run manage-stories         # End-of-sprint automation
See package.json for all available commands and team-specific variants.

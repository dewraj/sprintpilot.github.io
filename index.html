Quick Start

Prerequisites:
  • Node.js 18+
  • Azure Developer CLI (azd)
  • Access to Azure DevOps org/project
  • Windows + Outlook for email features
Install & Configure: 
git clone https://github.com/desing_microsoft/SprintPilot
cd SprintPilot
npm install
azd auth login
# Edit config.js as needed

Automate Planning & Hygiene Tasks

  • Detects current and next sprint for your team
  • Filters, moves, closes, and splits User Stories automatically
  • Writes structured JSON logs of sprint actions
  • Supports non-interactive automation for hands-free operation
npm run manage-stories        # End-of-sprint story hygiene

Risk Scoring Methodology

Priority, Age, Sprint Moves, Split Count, Risk Field, Target Date, Story Points, and Urgency are all weighted to compute risk.
Final risk level: Critical, High, Medium, Low, Minimal

Copilot Chat & MCP Integration

Plan and manage sprints interactively via Azure DevOps MCP and GitHub Copilot Chat.
Configure .vscode/mcp.json and ask Copilot Chat to run SprintPilot commands for you.
